## How to Use

1. **Extract** the archive to any folder  
2. **Run** `DeckTracker.exe` as Administrator  
3. **Select** your Slay the Spire 2 install directory  
4. **Choose** overlay style (Minimal / Full / Streamer Mode)  
5. **Launch** Slay the Spire 2 — the overlay appears automatically  
6. **Play** and track everything — review stats in the dashboard after each run

> **Pro tip:** Enable **Draw Probability** + **Card Advisor** together for maximum decision-making power during high Ascension runs.

## Troubleshooting

| Problem | Fix |
|---|---|
| Overlay not showing | Run as Administrator, check game path, disable other overlays |
| Draw probabilities wrong | Restart tracker before a new run, ensure no mid-run launch |
| Run not logging | Check runs/ folder permissions, ensure tracker launched before game |
| High CPU usage | Switch to Minimal overlay mode, reduce refresh rate in settings |
| Export failing | Check disk space, ensure runs/ folder is not read-only |
| Conflicting with other mods | Load DeckTracker last, or use standalone mode |
